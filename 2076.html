UN SINGE
Un singe entre dans un bar et demande au serveur : 
"m'sieur, vous avez des bananes?"
le serveur lui répond :"non".
le singe : "m'sieur, vous avez des bananes?"
le serveur : "non".
le singe : "m'sieur, vous avez des bananes?"
le singe : "m'iseur, vous avez des bananes?"
le singe : "m'sieur, vous avez des bananes?"
le serveur : "écoutez, si vous me demandez encore UNE fois si j'ai des bananes, je vous cloue le langue sur le bar!"
le singe : "vous avez des clous?"
le serveur : "non".
le singe : "m'sieur, vous avez des bananes?"
